ABOUT CUBRID

국산 RDBMS의 표준
CUBRID란 무엇인가

네이버가 실전 검증하고, 대한민국 공공기관이 가장 많이 선택한 엔터프라이즈 오픈소스 DBMS

INFO

CUBRID란?

CUBRID는 대한민국 NAVER에서 개발하고 현재 CUBRID Inc.가 운영하는 오픈소스 관계형 데이터베이스(RDBMS)입니다. 1997년부터 국내 대형 포털 서비스에서 수십억 건의 트랜잭션을 처리하며 엔터프라이즈 환경에서의 안정성을 증명했습니다. GPL v2.0 기반으로 소프트웨어 자체 비용이 없으며, Oracle DB를 대체하는 공공기관 표준 DBMS로 자리 잡고 있습니다.

WHY

왜 선택하는가

  • Oracle SQL / PL/SQL 완전 호환으로 재개발 비용 최소화
  • GPL v2.0 오픈소스 — 상용 라이선스 비용 제로
  • 내장 HA로 별도 클러스터 솔루션 불필요
  • 한국어 기술지원 · 국내 전문 커뮤니티 활성화
  • 공공기관 조달청 등록 · 전자정부 프레임워크 공식 지원
PLATFORM FEATURES

CUBRID 핵심 기능

엔터프라이즈 DBMS에 요구되는 모든 기능을 오픈소스로

DB
Oracle Compatibility

Oracle 완전 호환

Oracle SQL 문법, PL/SQL 패키지·함수·트리거, 데이터 타입을 동일하게 지원합니다. 기존 Oracle 애플리케이션의 코드 수정을 최소화하면서 이전할 수 있습니다.

HPC
High Performance

검증된 고성능 엔진

네이버 포털의 수십억 건 일 처리량을 실전 검증한 CUBRID 엔진. 동시 접속 처리, 멀티 프로세서 최적화, 인덱스 스킵 스캔 등 고성능 쿼리 최적화를 제공합니다.

HA
High Availability

고가용성 HA 내장

CUBRID HA(High Availability) 기능으로 Master-Slave 자동 페일오버를 지원합니다. 데이터 손실 없는 자동 전환으로 미션 크리티컬 환경에서도 서비스 연속성을 보장합니다.

SEC
Enterprise Security

엔터프라이즈 보안

테이블·컬럼 단위 접근 권한 제어, TLS/SSL 암호화 통신, 감사(Audit) 로그, 암호화 백업을 지원합니다. 공공기관 보안 요구사항을 완벽히 충족합니다.

STD
Open Standard

오픈 표준 기반

SQL-92, SQL-99 표준을 준수하며 JDBC, ODBC, PHP, Python, Node.js, .NET 등 다양한 언어 드라이버를 공식 지원합니다. 벤더 종속 없는 자유로운 개발 환경을 제공합니다.

TCO
Cost Optimization

라이선스 비용 혁신

GPL v2.0 기반 오픈소스로 소프트웨어 자체 비용이 없습니다. 기존 Oracle DBMS 대비 라이선스·유지보수 비용을 최대 70% 이상 절감합니다.
