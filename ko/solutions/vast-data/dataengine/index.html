VAST DataEngine

AI의 신경계

이벤트 트리거, Python 함수, Kafka 호환 브로커로 데이터에 가까운 서버리스 컴퓨트를 실현합니다.

핵심 기능

서버리스 데이터 처리의 새로운 기준

🐍

서버리스 Python 런타임

컨테이너화된 Python 함수를 작성하고 배포합니다. 인프라 관리 없이 데이터 처리 로직에 집중할 수 있습니다.

⚡

이벤트 기반 트리거

데이터 변경 시 자동으로 워크플로우를 시작합니다. 데이터 인리치먼트, 벡터화, 태깅, 라우팅 등을 자동화합니다.

📡

Kafka 호환 이벤트 브로커

네이티브 Kafka API 호환 브로커로 결정적 저지연과 선형 확장을 제공합니다. 외부 Kafka 클러스터가 필요 없습니다.

🎯

데이터에 가까운 컴퓨트

데이터를 이동시키지 않고 컴퓨트를 데이터에 가까이 배치합니다. 네트워크 오버헤드를 제거하고 처리 속도를 극대화합니다.

🚀

GPU 활용 최적화

DataSpace와 연동하여 데이터와 컴퓨트 배치를 조율합니다. 올바른 데이터가 준비된 상태에서 작업이 즉시 시작됩니다.

🔄

엔드투엔드 파이프라인

인제스트 → 처리 → 인리치먼트 → 저장까지 전체 데이터 파이프라인을 단일 플랫폼에서 오케스트레이션합니다.

데이터 파이프라인 자동화가 필요하신가요?

이벤트 기반 처리, 실시간 스트리밍, GPU 워크로드 오케스트레이션 — VWorks가 함께합니다.
